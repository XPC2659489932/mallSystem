## 大学生超市后台管理系统

## 技术栈

> #### React + React-Router + webpack + ES6 + BootStrap + Sass

## 项目运行

#### 注意：由于涉及大量的 ES6 新属性，node 需要 6.0 以上版本

>  1. git clone git@github.com:xingpengchao/mallSystem.git
>  
>  2. cd mallSystem 
>
>  3. npm install 
>
>  4. npm install yarn -g  (安装yarn)
>
>  5. yarn dev  (启动项目)
>
>  6. 浏览器打开 localhost:3200
>
>  7. yarn dist (项目打包)
